Основы HTML

HTML (HyperText Markup Language) - это язык разметки для создания веб-страниц. Он состоит из различных тегов, которые определяют структуру и отображение страницы.

Форматирование текста

Можно выделять текст разными способами:
Курсив
Полужирное начертание
Подчеркивание

Индексы и формулы

Важные математические формулы:
H2O - Формула воды
x2 + y2 = R - Уравнение окружности

Использование тега

Этот текст использует тег font для изменения шрифта, размера и цвета.
